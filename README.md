# The Cat App

## Requirements

* Node ^14.17.4

## Project setup
```
npm install
```

## Installation
1) Create an ```.env``` file from ```.env.example``` and set the Cat Api Key.

## Development

### Compiles and hot-reloads for development
```
npm run serve
```

The application will be available on:
```
http://localhost:8080/ 
```
